Resful Booker Automation Framework

This project uses Allure Report (Or Surefire report) to provide detailed insights into api test execution.

View Locally
Firstly, run the tests and generate the report on your local machine. follow these commands
1. Execute the test: mvn test
2. Download the artifact: allure-result
3. After downloading, goto the folder allure-result
4. Open the terminal (if you are using MacOS), type the command: allure serve allure-results
